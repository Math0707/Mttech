# Mttech Site

Projeto institucional moderno para a agência de marketing digital **Mttech**, desenvolvido com **Next.js (App Router)**, **React**, **TailwindCSS**, **GSAP** e **Framer Motion**.

## Como rodar

```bash
npm install
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`.

## Estrutura

- `app/` - rotas e layout principal
- `components/` - componentes reutilizáveis
- `sections/` - seções da landing page
- `styles/` - estilos globais
- `utils/` - helpers

## Recursos incluídos

- SEO básico via `metadata` do App Router
- Layout responsivo com menu mobile
- Microinterações com Framer Motion
- Scroll reveal com GSAP + ScrollTrigger
- Visual dark tech com gradientes roxo/azul
